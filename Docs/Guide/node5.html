1.5 The Inspector Window

The Inspector always shows the configuration of the current component. The Inspector contains 4 panes, the Constructor pane, the Properties pane, the Events Pane and the Objects Pane.

The Constructor pane allows you to edit properties which are required at object construction time, e.g. the name and style of the component. Look at the wxPython help for the values passed to the constructor for a control. The styles in the manual are of particular importance.

The Properties pane gives fine grained control to all properties for this component, e.g. fonts and colours.

The Events window allows you to select the events which you wish to handle in your code. The events are grouped into logical groups.

The Objects pane allows you to navigate through your components in an alternative format to the Designer pane. This is particularily useful if you have invisible or overlapping components.
