1.6 Managing the Application

When the module being edited is a application, the Editor provides a special Application View. The Application pane replaces the 'Explore' pane which is displayed for other modules. This Application View allows you to easily track the files in your application, and to add new modules, dialogs and other types of files to your application.

Files are added from the 'New' pane in the Palette. If you add a file with the Application View open, the file will automatically be added to the application.

The application view also provides access to the Profile tools.
